# Sentiment Analysis of Restaurant Reviews Case Study

## Overview
This case study explores how sentiment analysis can be used to understand customer feedback from restaurant reviews. Students will build a model that classifies review text as positive, neutral, or negative, and evaluate how well the model performs.

## Objective
The goal of this project is to:
- Clean and prepare text data  
- Convert text into numerical features using TF-IDF  
- Train a classification model  
- Evaluate model performance  
- Interpret results and identify limitations  

## Repository Structure

- **data/**  
  Contains the dataset used for the case study  

- **code/**  
  Contains the notebook used to build and evaluate the model  

- **materials/**  
  Contains supporting resources to help understand sentiment analysis  

- **hook_document.pdf**  
  Provides the scenario and context for the case study  

- **rubric.pdf**  
  Outlines the steps and expectations for completing the assignment  

## Instructions
Follow the rubric to complete the case study. Use the provided dataset and code as a starting point, and refer to the materials for additional support.

## Expected Outcome
Students will produce a working sentiment classification model and a short written explanation describing how the model performs and what insights can be drawn from the results.
