This folder contains supporting resources to help understand sentiment analysis.

## 1. Stanford Sentiment Analysis Slides
This resource provides an intuitive overview of sentiment analysis, including examples, applications, and a basic explanation of how sentiment classification works. It is intended to help build a foundational understanding of the topic.

## 2. Pang & Lee - Opinion Mining and Sentiment Analysis
This academic paper provides a more technical explanation of sentiment analysis, including how text is converted into features and how models are trained to classify sentiment. It offers deeper insight into the methods used in this case study.
